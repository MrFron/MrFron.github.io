Ejemplo 4 - Nos ponemos serios.
Ejemplo uso de métodos de objetos y el método sort - JavaScript
Mini programa que donde podremos calcular la media de notas de una clase y ver una lista ordenada sobre ellas.

Se trata de un programa que muestra el uso de métodos sobre objetos de JavaScript.
Métodos de números, cadenas de texto(Strings), Arrays y fechas, con el uso del método sort para hacer una lista con un orden personalizado (en este caso por nota descendiente).
El funcionamiento del programa consiste en añadir el nombre del alumno y su nota, presionaremos en el botón de añadir y se añadirá a un array, añadiremos cuantas queramos y una vez
le demos al botón de calcular nos calculará la media de las notas que hayamos introducido.
Para ver el ranking de notas tendremos que darle al botón de Mostrar Ranking y nos mostrará un ranking ordenado por la nota de cada alumno.
Este ejemplo es referente de la píldora FT-M06-CT4.P4.1 (part 1): Objectes del sistema.
